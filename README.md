# Welcome to stibride ! 
javafx project to get the shortest path between two Brussels metro stations.

# quick start
1. Clone this repository
```
git clone https://github.com/Gsewodo/stibride.git
```
2. run 
```
$ cd stibride/
$ mvn javafx:run
```
